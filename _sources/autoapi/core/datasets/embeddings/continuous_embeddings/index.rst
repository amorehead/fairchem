   CGCNN-like embeddings using continuous values instead of original k-hot.

   Properties:
       Group number
       Period number
       Electronegativity
       Covalent radius
       Valence electrons
       First ionization energy
       Electron affinity
       Block
       Atomic Volume

   NaN stored for unavaialable parameters.
